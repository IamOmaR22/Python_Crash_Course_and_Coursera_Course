Q-1. Which Python keyword indicates the start of a function definition?
Ans-  def


Q-2. In Python, how do you indicate the end of the block of code that makes up the function?
Ans-  You de-indent a line of code to the same indent level as the def keyword


Q-3.  In Python what is the raw_input() feature best described as?
Ans-  A built-in function


Q-4.  What does the following code print out? def thing(): print 'Hello'; print 'There'
Ans- There


Q-5. In the following Python code, which of the following is an "argument" to a function? x = 'banana'; y = max(x); print y
Ans- x


Q-6.  What will the following Python code print out? def func(x) : print x; func(10); func(20)
